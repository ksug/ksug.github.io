엔카닷컴
Spring Boot와 Event 기반 Materialized view를 이용한 Legacy DB 숨통 틔우기

작년 9~11월 엔카닷컴 전체 트래픽의 절반 이상을 차지하는 ‘차량 상세 페이지’의 저장소를 RDB에서 Event 기반 Materialized view로 전환하는 프로젝트를 진행 하였습니다.
그 결과 Legacy DB의 숨통이 확 트임은 물론 안정성, 성능, 확장성 측면에서도 매우 만족할 만한 결과가 있었기에 프로젝트를 제안하고 이끌면서 겪었던 경험과 노하우를 최대한 공유 드리려 합니다.
사용한 기술셋은 Spring Boot, Kafka, MongoDB, Ehcache, Resilience4j, Hiberante, Consul, AWS Cloud 등 입니다.

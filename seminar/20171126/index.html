JDK 9와 Java EE 8, 그리고 Spring 5...


2017년 9월,

Spring 5가 공개되었습니다. JDK 9과 Java EE 8 API 지원, 리액터(Reactor) 3.1과 Junit 5, 코틀린이 통합되었고
함수형 API 기반에 새로운 웹 프레임워크인 Spring WebFlux에 등장까지 큰 변화가 있었습니다.
JDK 9과 Java EE 8에도 Reactive Stream API(Flow API), HTTP/2 Client,
Servlet 4.0, JSON Binding API 등 다양한 기능들이 공개되었습니다.

JDK 9와 Java EE 8, 그리고 Spring 5에 새로운 모습들을 살펴보고자 합니다.

Java 9 new feature

이번 Java 9에 추가된 표준들과 표준의 구현으로 들어간 내용들을 소개하는 자리를 갖고자 합니다.

HTTP/2 세대의 Java

HTTP/2 스펙이 공표된지 2년이 넘었지만 이를 실무에서 적용하기에는 아직 난관이 있습니다. 이 발표에서는 Servlet 4.0과 Java9에서 HTTP/2를 지원하는 스펙들과 이를 실무에서 어떻게 적용할 수 있을지 고민한 과정을 공유합니다.

- HTTP/2 시대의 이상
- Java 생태계의 HTTP/2 지원
   - Servlet 4.0 : Server push
   - Java9 : HttpClient
- 현실의 제약과 극복
   - 웹서버, 어플리케이션 서버 구성
   - 클라이언트 모듈 선택
